# 50 Projects in 50 Days: Good, Fast, Cheap

This project was created by following the Udemy tutorial: [50 Projects In 50 Days - HTML, CSS & JavaScript](https://www.udemy.com/course/50-projects-50-days/?couponCode=24T3MT53024).

## Description

Select if you want a project that's good, fast, or cheap. You can't all have all 3! Attemping to pick all 3 will turn off one of the others.
